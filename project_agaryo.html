    Aga.Ryo game

    Client

    Myself

    Time

    Aug 2019

    Project overview

    For my 2019 baccalaureate project in computer science option, I had to make a game either in C or in js. The bac having been canceled I still carried out the project by myself in javascript on p5.js, a javascript library for coding online. Aga.Ryo is a parody of the game agar.io, which is played in a duel on the same pc. The goal is to eat the opponent, by being as big as possible by eating cells on the playing field. Bonuses appear as you go to make the game faster and more dynamic.
